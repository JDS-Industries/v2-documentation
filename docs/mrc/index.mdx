# MRC - michaels, ross and cole

MRC is a tool used on a daily basis for analyzing and maintaining data related to Customers, Current Daily Sales, Comparing monthly/yearly sale period, generating summaries based on multiple reports, scheduling email tasks, running nightly reports and other department related report. 

The table data is pulled from our IBM database into m-Power where it can be to create reports, web forms and other types of web applications. 

:::info

MRC can be divided in two parts:

1. m-Power: The software used to build reports
2. STAR: The CRM built used view all the reports

:::

Documentation can be found at their [`website`](https://www.mrc-productivity.com/mrcjava/docs)
