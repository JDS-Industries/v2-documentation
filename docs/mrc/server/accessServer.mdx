# Restarting or Stopping a MRC Server 

:::info

The Server is accessed from **VMware vSphere**. Visit [`Server`](https://10.1.2.69/ui/#?extensionId=vsphere.core.inventory.serverObjectViewsExtension&objectId=urn:vmomi:VirtualMachine:vm-61:8a4997c3-ca6e-45fa-bfb7-15e64d5cfb94&navigator=vsphere.core.viTree.hostsAndClustersView)

The login credentials can be found in the vendor info folder

:::

After logging in Click on `Launch Web Console` and Select `OK`

Click on `Select Ctrl+Alt+Delete` button in the top right corner and login using Administration account. 

Click on `Services` in Taskbar. The first two services should `Apache Tomcat 9.0 TC9DEV` and `Apache Tomcat 9.0 TC9PROD`. 

Click on the one you wish to Stop or Restart the server.
