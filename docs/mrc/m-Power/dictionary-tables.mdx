## Register a Table

All the database tables from IBM i are available in m-Power. To register a table:

Click on the <Icon inline icon="iconamoon:settings-fill"/> Admin in the top right corner.

Select `Tables` <Icon inline icon="zondicons:arrow-right" /> `Manage/Register Tables`.

If `Manage Tables` is selected, click on `Register Table` <Icon inline icon="zondicons:arrow-right" /> Select a library <Icon inline icon="zondicons:arrow-right" /> Select the table to register.

## Creating a Dictionary

A Dictionary is a work environment for m-Power users. It is a collection of tables, fields, and other objects that are used to create applications. For each Dictionary applications like `Retrievals`, `Reports`, `Summaries`, `Maintainers` are specific to that Dictionary.

Click on the <OptimizedImage publicId="v1693330340/v2/documenation/mrc/select_dictionary_u7wexe" /> next to the `Dictionary name | Home` in the top left corner.

Select `Switch Dictionary`.

Click on `Create New Dictionary`

Assigning a Dictionary

Follow Step 1 and 2 from the above section.

Click on `Assign Dictionary` <Icon inline icon="zondicons:arrow-right" /> Select the Dictionary to assign.
