# Add New User to Star

Log into m-Power and click on <icon inline icon="iconamoon:settings-fill"/> Admin in the top right corner.

Goto `Menu and Security` <icon inline icon="zondicons:arrow-right" /> `Application Users` <icon inline icon="zondicons:arrow-right" /> `Create New User`

Add their Username, Firstname, Lastname and Email address (Password doesnt need to be added). Then hit Save User.

Search for the username and Click on `Update User` <icon inline icon="zondicons:arrow-right" /> User Roles <icon inline icon="zondicons:arrow-right" /> Select a role to add. 

Assigning a role gives them access to only that menu in Star and hit Save User.
