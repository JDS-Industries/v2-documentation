# Add New User to Star

1. Log into m-Power and click on <Icon inline icon="iconamoon:settings-fill"/> Admin in the top right corner.

2. Goto `Menu and Security` <Icon inline icon="zondicons:arrow-right" /> `Application Users` <icon inline icon="zondicons:arrow-right" /> `Create New User`

3. Add their Username, Firstname, Lastname and Email address (Password doesnt need to be added). Then hit Save User.

4. Search for the username and Click on `Update User` <Icon inline icon="zondicons:arrow-right" /> User Roles <Icon inline icon="zondicons:arrow-right" /> Select a role to add. 

5. Assigning a role gives them access to only that menu in Star and hit Save User.
